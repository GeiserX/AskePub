# CLAUDE.md — AskePub

## Overview
Telegram bot that helps prepare ePubs by using ChatGPT-4o to write contextual study notes. Users upload an ePub, select chapters, and receive annotated ePubs, Word documents, or PDFs with AI-generated study notes.

## Tech Stack
- Python (pyproject.toml)
- OpenAI API (GPT-4o)
- Telegram Bot API
- Docker / Docker Compose
- pytest (tests/)

## Development
```bash
# Install dependencies
pip install -r requirements.txt

# Run tests
pytest tests/

# Run with Docker
docker compose up -d
```

Required environment variables: `TOKEN`, `OPENAI_API_KEY`, `ADMIN_ID`, `USER_IDS`.

## Architecture
- `src/` — main application source code
- `tests/` — test suite
- `locales/` — i18n translations (EN, ES, IT, FR, DE, PT, BG)
- `docs/` — documentation and images
- `Dockerfile` — container build definition
